# 图片变现系统 v1.0

> 清禾的图片变现系统，让图片能力变成钱。

---

## 系统架构

```
客户需求 → 接单渠道 → 需求整理 → 图像生成 → 交付 → 收款
```

## 2. 需求整理

### 图片类型
| 类型 | 价格区间 | 举例 |
|------|----------|------|
| 商业人像 | 50-200元/张 | 模特图、商业照 |
| 产品图 | 30-100元/张 | 电商主图、详情页 |
| 海报/封面 | 50-300元/张 | 宣传海报、社交媒体封面 |
| 头像/壁纸 | 10-50元/张 | 定制头像 |
| 游戏/角色图 | 100-500元/张 | 角色立绘、场景图 |

### 需求表单
```
客户名称：

```

---

## 3. 图像生成

### 清禾的能力
- ComfyUI + SDXL Turbo：高质量出图
- 参考图img2img：零畸形率95%+
- LLaVA审核：确保质量

### 工作流
1. 接收需求（文字/语音）
2. 分析需求→生成prompt
3. 生成图片
4. LLaVA审核
5. 如需修改，循环3-4
6. 交付

---

## 4. 交付标准

### 格式
- PNG/JPG
- 分辨率：1024x1024以上
- 如需高分辨率，额外报价

### 交付方式
- 微信/飞书发送原文件
- 百度网盘（大型文件）

### 售后
- 小修小改免费1次
- 大改需重新报价

---

## 5. 收款流程

### 定价参考
- 先收款再出图（防白嫖）

- 奕霖协助收款

### 收款方式
- 微信转账
- 支付宝
- 奕霖的账户代收
